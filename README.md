## Dynamics
* Help Santa save Christmas by collecting as many presents as possible. 
* Try to avoid the snowman
* Dynamics of a simple arcade game

## Mechanics
* Santa guidance
* Hovering gift
* Hovering snowman
* Collecting gifts
* Game over

* Gift
